# poc-vr-backend

## Installation

Clone the repo
   ```sh
   git clone git@github.com:CelianNoel/poc-vr-backend.git
   ```
Install the dependancies
   ```sh
   npm install
   ```
## Run the server

Run the express server
   ```sh
   node index.js
   ```

## Deployement

Deploy on heroku server
   ```sh
   git push heroku main
   ```
